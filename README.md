# Botj

J is a bot that helps you to automate your tasks. It is a desktop application built with Electron tested on Windows 10/11 and macOS.

## Development

### Install

```bash
npm install
```

### Run

```bash
npm start
```

## Start From Scratch

1. Generate from boilerplate by [electron forge](https://www.electronforge.io)

```bash
npm init electron-app@latest my-new-app -- --template=webpack-typescript
```

2. Create your own app icons by following the instructions in [Custom App Icons](https://www.electronforge.io/guides/create-and-add-icons).
